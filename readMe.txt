**②アサイン管理**

*未アサイン枠や空き要員を即時把握

必要ロールの充足状態を可視化

空きメンバーを条件検索で抽出

稼働過多・稼働不足も早期発見

*ガントチャートで直感的に稼働管理

案件横断でメンバー稼働を一覧、メンバーを複数案件に割り当てることができ、兼務状況も自動的に可視化されます。

兼務状況を時系列で把握

時間軸の移動もワンクリック

*スキル・経験に基づく最適配置

稼働率・期間・スキルなど複数条件から空き要員を抽出できます。次の案件に投入可能な人材をすぐに発見できます。契約で必要なロールと現在の配置を比較し、欠員枠のみを抽出します。アサイン漏れ防止に役立ちます。

経験値とロール適性から判断

配置変更による負荷の変化も把握



**③予実管理**

*月次予実管理　売上・原価・人件費を案件単位で月次管理。案件の黒字/赤字状況をひと目で確認できます。

*予算・見込・実績の対比

予算との差異や、月次見込の変動を把握し、プロジェクトの健全性を早期に判断できます。

*配賦自動化

売上・人件費を稼働割合に応じて自動配賦。手作業によるミスや負荷を大幅に削減します。

*アサイン変更の即時反映

アサイン変更がそのまま予実へ即時反映され、シミュレーションや意思決定を迅速化できます。

*カテゴリー

任意のカテゴリーでグルーピングして集計することで、様々な角度から分析できます。

*月次ロック　締め後の予実をロックすることで、正確な月次管理を支えます。

*アサイン・予実との連動　案件構造を基盤に、アサイン情報や予実管理と連動して管理精度を高めます。


**④案件・契約管理**

*契約内容管理　単価・契約形態・期間などの契約情報を体系的に管理し、同一案件内の複数契約でフェーズごとの管理もできます。

*必要ロール管理　案件ごとに必要な役割と人数を定義し、アサイン状況とのギャップを可視化できます。

*アサイン・予実との連動　案件構造を基盤に、アサイン情報や予実管理と連動して管理精度を高めます。

*自由なタブ/項目設定　受注ステータス、得意先などの単一項目だけでなく、案件状況メモなどの複数/時系列データも自由に設定できます。

*ビュー機能　一覧上で検索条件、ソート順、表示列を柔軟に設定し、それらを保存できるため、同じ検索条件を何度も入力いただく必要がありません。

*エクスポート/インポート　一覧に表示されているデータのエクスポート、及びインポートによる一括更新が可能です。



**⑤ノーコードカスタマイズ**

*タブ編集　タブを自由に追加・削除・並び替えでき、用途に応じた画面構成を構築できます。

*フィールド編集　テキスト・数値・選択肢・日付など、多様なフィールドを自由に追加できます。

*独自メニュー作成　得意先・取引先管理、売上メニューなど、自社専用のメニューを作成できます。

*表示制御・入力制御　権限グループごとに表示可否や入力制限を設けられ、誤操作や情報漏洩を防ぎます。

*ビュー機能　一覧上で検索条件、ソート順、表示列を柔軟に設定し、それらを保存できます。

*エクスポート/インポート　一覧に表示されているデータのエクスポート、及びインポートによる一括更新が可能です。
